## 📖 简介

**MochaNote（摩卡便签）** 是一款追求极致轻量与高审美的本地桌面便签工具。它以 11MB 的极小体积，提供了媲美高端笔记应用的富文本编辑、主题切换和文件管理体验。

所有数据均存储在本地，无需登录、无需联网、无需注册，即开即用，完全保护你的隐私。

---

## ✨ 功能特性

### 📝 富文本编辑

- **完整格式支持** — 标题（H1/H2/H3）、粗体、斜体、删除线、引用块、代码块
- **任务清单** — 可勾选的复选框列表，管理待办事项
- **图片粘贴** — 直接从剪贴板粘贴图片，或拖拽图片文件到编辑区域
- **气泡工具栏** — 选中文字即弹出格式按钮，操作直觉化
- **浮动菜单** — 空行自动弹出内容块快捷面板
- **内置模板** — 会议纪要、待办清单、工作日志，一键生成

### 📁 文件与组织

- **文件夹管理** — 创建、重命名、删除文件夹，自定义 8 种颜色标识
- **右键菜单** — 文件夹和笔记均支持右键操作：重命名、删除、更换颜色、复制副本、移动至文件夹
- **拖拽排序** — 拖拽笔记卡片调整显示顺序
- **即时搜索** — 标题+内容全文检索，毫秒级响应
- **笔记计数** — 文件夹和笔记列表实时显示数量

### 📤 数据导出

- **单篇导出** — 标题栏一键导出当前笔记为 Markdown (.md) 文件
- **批量导出** — 在设置中一键将所有笔记批量导出至指定文件夹
- **智能转换** — HTML 富文本自动转换为标准 Markdown 格式（标题/列表/任务/代码/图片）
- **原生保存对话框** — 使用系统原生文件选择器，操作符合直觉

### 🎨 四套精选主题

### 🪟 窗口体验

- **无边框透明窗体** — macOS 风格设计 + Windows Acrylic 亚克力材质
- **窗口置顶** — 便签固定在屏幕最上层，随时查阅

- **弹性动画** — Spring 物理曲线驱动的流畅过渡效果

## 🚀 使用方式

### 下载安装

1. 从发布页面下载 `MochaNote.exe`
2. 直接双击运行，无需安装

> **系统要求**：Windows 10/11 (x64)，需系统已安装 WebView2 运行时（Windows 10 21H2+ 及 Windows 11 已自带）

### 快速上手

| 操作 | 方式 |
|------|------|
| 创建笔记 | 侧边栏笔记区 `+` 按钮 |
| 切换主题 | 标题栏 ⚙ 按钮 → 主题选择 |
| 导出笔记 | 标题栏 ⬇ 按钮（当前笔记） |
| 批量导出 | ⚙ 设置 → 数据 → 批量导出 |
| 窗口置顶 | 标题栏 📌 按钮 |
| 文件夹操作 | 双击重命名，右键更多选项 |
| 笔记操作 | 右键笔记卡片 → 复制/移动/导出/删除 |
| 插入图片 | 编辑区域直接 Ctrl+V 粘贴 |
| 拖拽排序 | 拖动笔记卡片调整位置 |

### 数据存储

所有笔记数据存储在用户本地：
```
%USERPROFILE%/.mochanote/    （应用配置）
浏览器 localStorage           （笔记内容）
```

## ❓ 常见问题

**Q: 启动后窗口空白？**
> 请确认系统已安装 Microsoft Edge WebView2 运行时。在 Windows 11 中已默认预装。

**Q: 数据如何备份？**
> 使用 设置 → 批量导出 功能，可将所有笔记导出为 Markdown 文件至任意目录。

**Q: 如何彻底退出？**
> 点击窗口右上角的红色按钮（●）即可退出程序。
