# MochaNote（摩卡便签）

一款简洁优雅的本地便签应用，专注于简洁与高效的笔记体验。

## 功能特性

- **便签管理** - 创建、编辑、删除便签，支持标题和富文本内容
- **文件夹分类** - 使用文件夹组织便签，支持自定义分类
- **搜索功能** - 快速搜索便签标题和内容
- **窗口置顶** - 一键置顶/取消置顶窗口，随时快速记录
- **主题切换** - 4 种精美主题风格
  - Alabaster Light - 纯净明亮
  - Warm Paper - 温暖舒适
  - Midnight Charcoal - 深邃专注
  - Soft Sage - 清新自然
- **富文本编辑** - 支持标题、粗体、斜体、代码、列表、任务列表
- **模板功能** - 内置会议纪要、待办事项、工作日志模板
- **系统托盘** - 最小化到系统托盘，后台运行
- **本地存储** - 数据安全存储在本地，无需联网

## 技术栈

| 技术             | 说明     |
| -------------- | ------ |
| React 19       | 前端框架   |
| Vite 8         | 构建工具   |
| Electron 41    | 桌面应用框架 |
| Zustand        | 状态管理   |
| Tiptap 2       | 富文本编辑器 |
| Tailwind CSS 4 | 样式框架   |
| Framer Motion  | 动画库    |

## 快速开始

### 环境要求

- Node.js 18+
- npm 9+

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建应用

```bash
npm run build
```

## 项目结构

```
MochaNote/
├── electron/
│   ├── main.js        # Electron 主进程
│   └── preload.cjs    # 预加载脚本
├── src/
│   ├── components/
│   │   └── Editor.jsx # 富文本编辑器组件
│   ├── store/
│   │   └── useStore.js # Zustand 状态管理
│   ├── App.jsx        # 主应用组件
│   ├── App.css        # 应用样式
│   ├── index.css      # 全局样式和主题变量
│   └── main.jsx       # 入口文件
├── public/
│   └── icon.png       # 应用图标
├── index.html         # HTML 模板
├── package.json       # 项目配置
└── vite.config.js     # Vite 配置
```

## 界面预览

应用采用无边框透明窗口设计，支持 macOS 风格的交通灯按钮，界面简洁美观。

### 快捷操作

- 选中文本时显示格式化工具栏
- 空行时显示快捷插入菜单
- 支持键盘快捷键操作

## 数据存储

应用使用 Zustand 的 persist 中间件，数据自动保存在浏览器的 localStorage 中，刷新页面或重启应用后数据不会丢失。
